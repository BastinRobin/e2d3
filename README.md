E2D3
====

E2D3 is JavaScript library for using D3.js on Excel.

## Setting Up Development Environments

### Install Node.js

[Node.js](http://nodejs.org/)

### Fetch Dependencies

```shell
bin/setup
```

### Run Web Server for Development
```shell
bin/www
```
